## 🎨 Design System

### Typography
- **Font**: 'Outfit', system fonts
- **Sizes:**
  - Display: 96px (desktop), 60px (mobile)
  - H1: 80px → 64px (responsive)
  - H2: 60px → 48px
  - H3: 32px → 24px
  - Body: 18px → 16px

### Spacing System
- Consistent padding/margin ratios
- Responsive breakpoints
- Touch-friendly button sizes (44px+ minimum)

### Shadow System
- `shadow-elegant`: Light subtle shadow
- `shadow-card-hover`: Deeper shadow for interaction
- `shadow-glow-orange/cyan/green`: Colored glows

### Border Radius
- Small: 8px
- Medium: 12px
- Large: 16px
- Extra Large: 24px

### Animations Timing
- Quick: 0.2s - 0.3s
- Standard: 0.6s - 0.8s
- Slow: 2s - 6s (continuous)
